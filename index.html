關於我

  • 學歷: 屏東大學資訊工程學系一年級
  • 個性: 堅持、積極、樂觀
  • 興趣： 運動（健身、打棒球）、寫程式、聽音樂、打爵士鼓

    • 學習心得

    在屏東大學就讀的這一年間，在高中畢業對程式只有一點點概念的我到現在會傳址、結構體、動態宣告記憶體….還有網頁前端的設計像是HTML、CSS、JavaScript…..等好多是我之前從未接觸過的，但在大一的整年，我覺得我學到了很多，我很慶幸高中時期的我堅持了我的興趣而選擇了資訊工程，更開心的是讀完了一年，我很肯定這就是我想學習的。令我印象深刻的是程式設計的這門課程，教授的評分方式，只要輸出結果有任何的不對（像是忘記大寫、空格或是句號…..）一律是零分。剛開始我非常的不理解，覺得只要邏輯和輸出有答對就好，其他不用太在意，但教授上課時的幾句話點醒了我，他說：「如果客戶要求你的結果是這樣，結果因為你的疏忽或小差錯，可能會有嚴重的後果…….」我明白他想要從一些小地方要求我們程式要寫到非常完美，不要寫「有時候對」的程式，所以在這樣的情況下，我養成了注重細節的習慣，也讓打出來的程式更加的堅固，徹底的改掉了我「這樣就好」的個性，因此，我相信在屏東大學學習的這一年中，有紮實穩固底子的我能應付接下來的課業。（實作練習請見作品集）

    轉學動機

    想轉入彰師大的原因如下:

    貴校有很多姊妹校，提供學生管道能取得雙聯學位，所以我研究所有計畫繼續在貴校進修。

    貴校對於工科的發展性高，跟屏東大學不同的是，除了資訊科，彰師大還有像是電機、電子、機電….等科系，對於想加廣自己實力的學生，有更多的課程去修。

    本身是彰化人，想要讀附近的學校，所以貴校是最佳選擇。

    讀書計畫

    我目前就讀於國立屏東大學資訊工程學系一年級，從大學開始，我便確立了自己對資訊實作與系統開發的濃厚興趣，並積極投入實務專題與課外專案開發。為了朝著成為一位「能教學、能實作、能應用」的資訊人才邁進，我希望轉學至師資培育與教學環境優良的彰化師範大學，進一步深化我的專業知識與應用能力。

    在課程方面，我將針對以下幾個面向設定進修方向：

    • 程式設計與演算法：我會持續加強資料結構與演算法能力，精熟 C 語言與 Python，並透過解題平台提升問題解決技巧。
    • 網頁與應用系統開發：未來希望能在師大的教學環境中更深入學習系統分析與網頁技術（HTML/CSS/JavaScript）。
    • 人工智慧與資料分析：我正透過自學 Python 數據分析模組（如 pandas、numpy），並希望能進一步修習 AI 相關選修課，拓展視野。

    二、時間規劃與學習方法

    我使用 Notion 與 Git 管理每日任務與專案進度，每週安排固定時段複習課程內容、進行程式練習或參與開發。期望在學業與實作之間取得良好平衡。

    四、未來志向與進修方向

    未來我希望能持續升學資訊教育相關碩士班，並將資訊技能與教育結合，無論是擔任教師、助教或參與教育科技專案，都能讓我實踐「學以致用、教以致學」的理想。

    我相信，彰化師範大學資訊工程（或資訊教育）系重視「教學與實務並重」的課程設計，正好與我的個人學習目標與長期職涯志向相符。在貴校的資源與師資指導下，我將能更完整地成長為一位具備實務能力與教育熱情的資訊人才。

    技能

    • HTML / CSS / JavaScript
    • C / C#
