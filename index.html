關於我

  • 學歷: 屏東大學資訊工程學系一年級
  • 個性: 堅持、積極、樂觀
  • 興趣： 運動（健身、打棒球）、寫程式、聽音樂、打爵士鼓

    • 我從小對科技產品充滿好奇，喜歡研究電腦操作與軟體應用，也對人工智慧充滿好奇。這樣的興趣促使我選擇就讀國立屏東大學資訊工程學系。在學習的過程中，我對程式設計、系統架構與演算法等領域產生了濃厚的興趣，並開始思考如何將這些技術應用在更廣泛的層面，解決實際生活中的問題。

    在大學期間，我努力培養邏輯思維與實作能力，也積極參與課程報告與小型專題製作。然而，在學習過程中我發現校內缺乏硬體相關的課程與資源，使我在深入探索嵌入式系統、電路設計等實作層面時，感到受限。因此，我萌生了轉學的想法，期望能進入一個資源更完整、學習視野更開闊的環境繼續成長。

    我選擇申請國立彰化師範大學資工系，主要因為貴系擁有完整的課程架構，除了紮實的軟體訓練外，還能接觸到電機、電子、機電等相關領域，這樣的跨域學習正是我所追求的方向。此外，彰師大設有國際雙聯學位制度，也提供了我未來拓展學術視野與國際交流的可能性。再者，彰師地理位置接近我的家鄉，能讓我在兼顧家庭的同時，更專注於學業發展。

    我未來希望能報考研究所，繼續精進於人工智慧、系統設計或嵌入式應用等領域，並期望將所學知識實際應用於產業之中。若能順利進入貴系，我將珍惜這份學習機會，持續充實自我、發揮所長，朝著成為具有實作能力與創新思維的資訊人才邁進。

    學習心得

    在屏東大學就讀的這一年間，高中畢業對程式只有一點點概念的我到現在會傳址、結構體、動態宣告記憶體….還有網頁前端的設計像是HTML、CSS、JavaScript…..等好多是我之前從未接觸過的，但在大一的整年，我學到了很多，很慶幸高中時期的我堅持了我的興趣而選擇了資訊工程，更開心的是讀完了一年，我很肯定這就是我想學習的。令我印象深刻的是程式設計的這門課程，教授的評分方式，只要輸出結果有任何的不對（像是忘記大寫、空格或是句號…..）一律是零分。剛開始我非常的不理解，覺得只要邏輯和輸出有答對就好，其他不用太在意，但教授上課時的幾句話點醒了我，他說：「如果客戶要求你的結果是這樣，結果因為你的疏忽或小差錯，可能會有嚴重的後果…….」我明白他想要從一些小地方要求我們程式要寫到非常完美，不要寫「有時候對」的程式，所以在這樣的情況下，我養成了注重細節的習慣，也讓打出來的程式更加的堅固，徹底的改掉了我「這樣就好」的個性，因此，我相信在屏東大學學習的這一年中，有紮實穩固底子的我能應付接下來的課業。（實作練習請見作品集）

    轉學動機

    我希望從國立屏東大學轉學至國立彰化師範大學資工系，主要基於資源整合、學習規劃與未來升學的考量。

    首先，彰化師大在資工相關領域具備更完整且多元的資源，不僅有豐富的實驗設備，也開設了涵蓋電機、電子、機電等跨領域課程， 讓我在修課選擇上更具彈性，能從中獲得更全面的專業能力。相較之下，目前屏東大學以軟體課程為主，缺乏硬體相關的訓練， 這使我在學習過程中感到受限。

    此外，彰師資工系亦有與國際大學合作的雙聯學位制度，這讓我未來能有更多元的學術視野與國際發展機會。 而且彰師大地處中部，離我的家鄉更近，有助於兼顧學業與家庭支持系統，能讓我在更穩定的環境下專注於學習與研究。

    我未來有意報考研究所，而彰師大在資工與工程領域的研究風氣濃厚，若有機會進入貴系就讀，將為我未來的研究深造之路奠定更扎實的基礎。

    基於以上理由，我誠摯希望能有機會轉學至貴系，拓展更深更廣的學習領域，充實自我，為未來做好準備。

    讀書計畫

    我目前就讀於國立屏東大學資訊工程學系一年級，從大學開始，我便確立了自己對資訊實作與系統開發的濃厚興趣，並積極投入實務專題與課外專案開發。為了朝著成為一位「能教學、能實作、能應用」的資訊人才邁進，我希望轉學至師資培育與教學環境優良的彰化師範大學，進一步深化我的專業知識與應用能力。

    在課程方面，我將針對以下幾個面向設定進修方向：

    • 程式設計與演算法：我會持續加強資料結構與演算法能力，精熟 C 語言與 Python，並透過解題平台提升問題解決技巧。
    • 網頁與應用系統開發：未來希望能在師大的教學環境中更深入學習系統分析與網頁技術（HTML/CSS/JavaScript）。
    • 人工智慧與資料分析：我正透過自學 Python 數據分析模組（如 pandas、numpy），並希望能進一步修習 AI 相關選修課，拓展視野。

    時間規劃與學習方法

    我使用 Notion 與 Git 管理每日任務與專案進度，每週安排固定時段複習課程內容、進行程式練習或參與開發。期望在學業與實作之間取得良好平衡。

    未來志向與進修方向

    未來我希望能持續升學資訊教育相關碩士班，都能讓我實踐「學以致用、教以致學」的理想。

    我相信，彰化師範大學資訊工程（或資訊教育）系重視「教學與實務並重」的課程設計，正好與我的個人學習目標與長期職涯志向相符。在貴校的資源與師資指導下，我將能更完整地成長為一位具備實務能力與教育熱情的資訊人才。
